# Claude Code Usage Analytics Platform

This project is designed to provide a comprehensive analytics platform for monitoring and analyzing telemetry data generated by the Claude Code application. It utilizes FastAPI for the backend and Streamlit for the frontend, allowing for efficient data handling and interactive visualizations.

## Project Structure

```
claude-code-analytics
├── backend                # FastAPI backend application
│   ├── app                # Main application package
│   ├── requirements.txt    # Backend dependencies
│   └── .env.example        # Example environment configuration
├── frontend               # Streamlit frontend application
│   ├── requirements.txt    # Frontend dependencies
│   └── .streamlit         # Streamlit configuration
├── scripts                # Scripts for database initialization and data loading
│   ├── init_db.py         # Initializes the PostgreSQL database
│   └── load_data.py       # Loads telemetry data into the database
├── docker-compose.yml      # Docker Compose configuration for services
├── .gitignore             # Git ignore file
└── README.md              # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd claude-code-analytics
   ```

2. **Backend Setup:**
   - Navigate to the `backend` directory.
   - Create a virtual environment and activate it:
     ```bash
     python -m venv venv
     source venv/bin/activate  # On Windows use `venv\Scripts\activate`
     ```
   - Install the required dependencies:
     ```bash
     pip install -r requirements.txt
     ```

3. **Frontend Setup:**
   - Navigate to the `frontend` directory.
   - Install the required dependencies:
     ```bash
     pip install -r requirements.txt
     ```

4. **Database Configuration:**
   - Copy the `.env.example` file to `.env` and update the database connection settings as needed.

5. **Initialize the Database:**
   - Run the database initialization script:
     ```bash
     python scripts/init_db.py
     ```

6. **Load Data:**
   - Load telemetry data into the database:
     ```bash
     python scripts/load_data.py
     ```

7. **Run the Applications:**
   - Start the FastAPI backend:
     ```bash
     uvicorn backend.app.main:app --reload
     ```
   - Start the Streamlit frontend:
     ```bash
     streamlit run frontend/app.py
     ```

## Architecture Overview

- **Backend:** The backend is built using FastAPI, providing a RESTful API for handling telemetry data. It connects to a PostgreSQL database for data storage and retrieval.
- **Frontend:** The frontend is developed using Streamlit, allowing users to interact with the telemetry data through a web interface. It includes various pages for dashboards, event exploration, and analytics.

## Dependencies

- FastAPI
- SQLAlchemy
- PostgreSQL drivers
- Streamlit
- Visualization libraries (e.g., Matplotlib, Plotly)
